# Train Management System

Hệ thống quản lý đường sắt - demo môn Phân tích thiết kế hệ thống.

## Mô tả

Dự án demo xây dựng bằng Java NetBeans, phục vụ mô phỏng các chức năng chính của hệ thống quản lý đường sắt.

## Các module chính

- Quản lý bán vé
- Quản lý lịch trình
- Quản lý tàu
- Quản lý nhân viên
- Thống kê thu chi

## Cấu trúc dự án

```text
src/                    Source code chính
nbproject/              Cấu hình NetBeans
build.xml               File build Ant
manifest.mf             Manifest của project
```

## Ghi chú

Không đưa các thư mục sinh tự động như `build/`, `dist/`, `nbproject/private/` lên GitHub.
